Практика алгоритмов на Codewars:

Задача: написать функцию, которая принимает список, содержащий неотрицательные целые числа и строки, и возвращает новый список только с числами, отфильтровывая строки.

Пример:

filter_list([1,2,'a','b']) == [1,2]
filter_list([1,'a','b',0,15]) == [1,0,15]

Решение:

 
                 filter_list(l) {
                    let result = [];
                    for (let item of l) {
                        if (typeof item === 'number') {
                        result.push(item)
                        }
                    }
                    return result
                }
                console.log(filter_list([1,2,'a','b']))

                // Test Results:
                Log
                [ 1, 2 ]
