Algorithm practice on Codewars:

Task: write a function that takes a list of non-negative integers and strings, and returns a new list with only the numbers, filtering out the strings.

Example:

filter_list([1,2,'a','b']) == [1,2]
filter_list([1,'a','b',0,15]) == [1,0,15]

Solution:

 
                filter_list(l) {
                    let result = [];
                    for (let item of l) {
                        if (typeof item === 'number') {
                        result.push(item)
                        }
                    }
                    return result
                }
                console.log(filter_list([1,2,'a','b']))

                Log
