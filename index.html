Algorithm practice on Codewars:

Task:
write a function that takes a list of non-negative integers and strings, and returns a new list with only the numbers, filtering out the strings.


Solution:

 
    filter_list(l) {
        let result = [];
            for (let item of l) {
                if (typeof item === 'number') {
                result.push(item)
                }
            }
            return result
        }
